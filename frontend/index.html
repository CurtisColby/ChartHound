ChartHound needs to translate your media server's file paths to the Docker /music mount so it can tag your physical files.

Media server sees: /media/colby/NAS1/MUSIC TAGGED/ABBA/Gold/Dancing Queen.flac

The gold part stays the same — only the prefix changes. Your prefix is everything before the artist folder.
